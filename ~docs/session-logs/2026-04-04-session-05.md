# Session Log - 2026-04-04 Session 05

## Summary

Architecture separation refactoring for Poppy compiler — Epic #238 Phase 1. Implemented the `ITargetProfile` strategy pattern to replace scattered if/else dispatch chains throughout CodeGenerator and SemanticAnalyzer.

## Work Done

### New Architecture Layer (`Poppy.Core.Arch`)

Created core abstractions:

- `IInstructionEncoder` — Architecture-neutral instruction encoding interface with `TryEncode()` and `IsBranchInstruction()`
- `EncodedInstruction` — Record struct (Opcode, Size) as the profile-level encoding result
- `IRomBuilder` — ROM building abstraction interface
- `ITargetProfile` — Composite profile providing Architecture, Encoder, DefaultBankSize, GetBankCpuBase, LongDirectiveSize, AdjustAddressingMode, CreateRomBuilder
- `TargetResolver` — Centralized string-to-enum target mapping with `Resolve()`, `GetDisplayName()`, `GetProfile()` factory

### 11 Target Profile Implementations

Each wraps the existing static `InstructionSet*` classes via adapter pattern:

- `Mos6502Profile` (NES) — wraps InstructionSet6502
- `Mos6507Profile` (Atari 2600) — wraps InstructionSet6507
- `Mos65sc02Profile` (Lynx) — wraps InstructionSet65SC02, includes INC/DEC→Accumulator adjust
- `Wdc65816Profile` (SNES) — wraps InstructionSet65816, LongDirectiveSize=3
- `Sm83Profile` (Game Boy) — wraps InstructionSetSM83, bank-conditional CPU base
- `Huc6280Profile` (TG-16) — wraps InstructionSetHuC6280
- `M68000Profile` (Genesis) — wraps InstructionSetM68000
- `Z80Profile` (SMS) — wraps InstructionSetZ80
- `V30mzProfile` (WonderSwan) — wraps InstructionSetV30MZ
- `Arm7tdmiProfile` (GBA) — placeholder encoder, real IsBranch
- `Spc700Profile` — placeholder encoder, real IsBranch

### CodeGenerator Dispatch Replacement

Replaced 6 dispatch chains with profile delegation:

1. `TryGetInstructionEncoding()` — 9-way if/else (~80 lines) → 6-line profile delegation
2. `IsBranchInstruction()` — 7-way if/else (~30 lines) → single-line delegation
3. `GetDefaultBankSize()` — 6-way switch → `_profile.DefaultBankSize` (SNES override kept for map mode)
4. `GetBankCpuBase()` — 5-way switch → `_profile.GetBankCpuBase(bank)`
5. `.long` directive size — ternary → `_profile.LongDirectiveSize`
6. 65SC02 accumulator adjust — inline check → `_profile.AdjustAddressingMode()`

### SemanticAnalyzer Refactoring

- `HandleTargetDirective()` — replaced 12-case inline switch with `TargetResolver.Resolve()`
- `SupportsAddressingMode()` — replaced 9-way switch with profile encoder delegation

### CLI Refactoring

- `--target` parsing — replaced 12-case inline switch with `TargetResolver.Resolve()`
