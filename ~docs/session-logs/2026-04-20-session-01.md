## Work Done

### Poppy #333 - NES Parser 6x Slower Than SNES (Closed)

**Root Cause:** The NES example file (`examples/nes-hello-world/main.pasm`) used ASAR-style syntax instead of Poppy syntax, causing 19 `ParseException` throws per compilation. Each exception allocates a full stack trace (~5μs), totaling ~95μs of pure exception overhead.

**Syntax Issues Fixed:**

- `.system:nes` → `.system nes` (colon not valid)
- `%wait_vblank` → `@wait_vblank` (wrong macro prefix)
- `.clear_ram:` → `@clear_ram:` (dot-labels are directives)
- `%ppu_addr` → `@set_ppu_addr` (wrong prefix and name)

**Results:**

| Metric | Before | After | Improvement |
|--------|--------|-------|-------------|
| Parser NES | 130 μs | 17 μs | **7.6x** |
| Full Compile NES | 216 μs | 101 μs | **2.1x** |
