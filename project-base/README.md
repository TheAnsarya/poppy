# Retro Game Project

This is a template project for building retro games with Poppy.

## Project Structure

```text
.
├── src/           # Source code files
├── assets/        # Game assets
│   ├── sprites/   # Sprite graphics
│   ├── sounds/    # Sound effects
│   └── music/     # Background music
└── build/         # Build output directory
```

## Getting Started

1. Edit the source files in `src/` to create your game
2. Add your game assets to the `assets/` directories
3. Build your project using the Poppy compiler

## Building

Run the build command to compile your game:

```bash
make
```

Your compiled game will be in the `build/` directory.
